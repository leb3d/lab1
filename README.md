## Лабораторна робота №1

**Рефакторинг програмного коду**

У роботі виконано рефакторинг програмного коду та перевірено структуру проєкту.
Код розподілено на окремі модулі, що спрощує його читання, підтримку та запуск.

### Структура

* `lab5.py` — окремий модуль
* `lab6.py` — окремий модуль
* `lab7.py` — окремий модуль
* `main.py` — головний файл запуску програми

У роботі використано принципи **модульності**, **структурування коду** та **розподілу функціоналу між файлами**.
